

# goooder

## install

### 1. install go module

```bash
go get github.com/version-1/goooder
```

### 2. add .env file

```bash
touch .env
```

set "DATABASE_CONNSTR" in .env file

folling is an example of DATABASE_CONNSTR
```bash
DATABASE_CONNSTR=postgres://postgres:password@127.0.0.1:54321/example?sslmode=disable
```
